DESCRIPTION:
------------
This module provides a menu reference field type.


REQUIREMENTS:
-------------
menufield.module requires the Field module.


INSTALLATION:
-------------
1. Place the entire menufield directory into your Drupal modules/
   directory.

2. Enable the menufield module by navigating to:

     administer > modules


Features:
---------
* one widget that lists modules present in the menu_custom table
* one display formatter that uses the default output of theme('link',...)
* option to choose no menu to display
